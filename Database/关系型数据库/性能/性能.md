QPS：Queries Per Second，每秒查询数
    定义：指数据库每秒能处理的 SQL 查询请求总数，这里的 “查询” 是广义的，包含所有 SQL 语句（SELECT查询、INSERT写入、UPDATE更新、DELETE删除等）。
    核心作用：反映数据库的瞬时并发处理能力，常用于评估读密集型业务的性能上限（比如电商商品列表查询、新闻资讯浏览）

TPS：Transactions Per Second，每秒事务数
    定义：指数据库每秒能执行的事务数量，事务是一组原子性的 SQL 操作（要么全部执行成功，要么全部回滚，符合 ACID 特性）。
    核心作用：反映数据库的事务处理能力，专门针对写密集型 / 混合读写的业务场景（比如电商下单、支付转账、用户注册）。
