## git mv命令
说明：移动或重命名文件、目录、符号链接

### 用法
```
git mv [-v] [-f] [-n] [-k] <source> <destination>
git mv [-v] [-f] [-n] [-k] <source> ... <destination directory>
```

| 选项 | 说明
| --- | ---
|  | 
|  | 
|  | 

### 示例
```sh
# 重命名README为README.md
git mv README README.md

# 移动file1、file2到dir目录下
git mv file1 file2 dir
```
