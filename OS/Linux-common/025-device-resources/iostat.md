## iostat命令
说明：报告 CPU 以及设备和分区的输入/输出统计信息

### 用法
```
iostat [ -c ] [ -d ] [ -h ] [ -k | -m ] [ -N ] [ -s ] [ -t ] [ -V ] [ -x ] [ -y ] [ -z ]
    [ --dec={ 0 | 1 | 2 } ] [ { -f | +f } directory ] [ -j { ID | LABEL | PATH | UUID | ... } ]
    [ -o JSON ] [ [ -H ] -g group_name ] [ --human ] [ --pretty ] [ -p [ device[,...] | ALL ] ]
    [ device [...] | ALL ]  [ in‐terval [ count ] ]
```

| 选项 | 说明
| --- | ---
|  | 
| -x | 显示详细的统计信息
| -y | 如果有多份报告，省略第一组报告
