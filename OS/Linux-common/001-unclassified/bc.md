## bc命令
说明：命令行计算器

### 用法
```
bc [ -hlwsqv ] [long-options] [  file ... ]
```

| 选项 | 说明
| --- | ---
| -i    | 交互模式，每次执行一条语句
| -l    | 加载标准数学库，提供更多的数学函数

### 示例
```sh
# 进入交互模式
bc -i   # 输出quit可以退出


echo "1+2" | bc             # 加法运算，返回3

echo "5-3" | bc             # 减法运算，返回2

echo "100*3" | bc           # 乘法运算，返回300
echo "123.456 * 10" | bc    # 乘法运算，返回1234.560

echo "100/3" | bc           # 除法运算，返回33。默认只保留整数部分
echo "100/3" | bc -l        # 除法运算，返回33.33333333333333333333。默认最多保留20位小数

echo "100%3" | bc           # 取余运算，返回1


# 计算百分比，保留两位
echo "scale=2;(310 - 500) * 100 / 500" | bc

```
