## pidof命令
说明：通过程序名称查找进程ID

pgrep是pidof的替代工具，功能更加强大。
但是在一些旧版本的系统中，可能没有pgrep命令，如果仅需要查找进程ID，那么建议使用pidof命令，它简单直接。

### 用法
```
xxx [选项]
```

| 选项 | 说明
| --- | ---
| -d    | 指定显示进程ID的分隔符，默认是空格
| -o    | 忽略指定进程ID

### 示例
```sh
# 查看nginx进程ID，默认就是精确匹配
pidof nginx -d ,
# 1550,1549,1426

# 忽略指定进程ID
pidof -o 1549 nginx
# 1550 1426
```
