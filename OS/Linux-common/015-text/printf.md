## printf命令&
说明：格式化并打印数据

### 用法
```
printf FORMAT [ARGUMENT]...
printf OPTION
```

| 选项 | 说明
| --- | ---
| -v VARIABLE | 将输出分配给变量VARIABLE，而不是打印到标准输出

| 格式符 | 说明
| --- | ---
| \n    | 换行符
| \\"   | 双引号
| \\\   | 反斜杠
| %%    | 百分号
| %s    | 字符串
| %d    | 有符号十进制整数
| %f    | 浮点数（默认6位小数）
| %u    | 无符号十进制整数
| %o    | 无符号八进制整数
| %x    | 无符号十六进制整数（小写）
| %X    | 无符号十六进制整数（大写）

### 示例
```sh
# 数字补零
# 语法：%0[宽度]d（仅对整数生效）
printf "%06d\n" 123             # 输出：000123

# 保留2位小数
printf "%.2f\n" "3.14159"       # 输出：3.14

# 右对齐，宽度为20
printf "[%20s]\n" "abcd"        # 输出：[               abcd]

# 添加方括号，不算方括号的宽度是20
printf "[%-20s]\n" "abcd"       # 输出：[abcd               ]

# 转换成十六进制
printf "%x\n" 255               # 输出：ff

# 用其他字符填充而不是默认的空格。本身不支持，需要通过管道和tr命令实现
printf "[%20s]\n" "abcd" | tr ' ' '-'   # 输出：[---------------abcd]
```
